¿Cómo crear una fecha y hora aleatoria en Python?

Última actualización: 25 septiembre 2023

El hecho de poder crear una fecha aleatoria nos da ventajas en algunos campos y aplicaciones de la programación, conocer cómo crearla nos servirá en aplicaciones como pruebas , análisis de datos, juegos, pruebas de rendimiento, seguridad, entre otras.

Para generar una fecha aleatoria en Python realmente es sencillo y su código es corto. Lo primero es importar las librerías necesarias, la primera es random, esta nos permite generar números aleatorios; la segunda librería es datetime, esta permite trabajar con fechas y horas.

Con el siguiente código obtendremos una fecha y hora aleatoria, la fecha se genera entre la fecha de inicio y la fecha final,estas variables inicio y final pueden ser cambiadas por las fechas entre las cuales se desee tener el rango.

import random as rd
from datetime import datetime

inicio = datetime(2020,1,1)
final =  datetime(2022,11,11)
random_date = inicio + (final - inicio) * rd.random()
dia=str(random_date.day)
mes=str(random_date.month)
anio=str(random_date.year)
hora=str(random_date.hour)
minuto=str(random_date.minute)
if len(dia)==1:
  dia="0"+dia
if len(mes)==1:
  mes="0"+mes
if len(hora)==1:
  hora="0"+hora
if len(minuto)==1:
  minuto="0"+minuto

print("Fecha: "+dia+"/"+mes+"/"+anio)
print("Hora: "+hora+":"+minuto)

Al final, el programa imprime por consola una fecha aleatoria entre las 2 fechas deseadas y una hora con minutos aleatorios. Así se vería al correr el código; seguramente al correr tu código no dará el mismo resultado ya que son fechas y horas aleatorias. Con los condicionales if que hay en el código nos aseguramos de que si el día, el mes, la hora o los minutos son menores a 10 se muestren con 2 dígitos y no con 1 como viene predeterminado en la librería.
