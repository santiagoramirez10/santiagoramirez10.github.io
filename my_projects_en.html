Programming projects

Identity matrix

In this project, programming is done in Python for the creation of an identity matrix of nxn dimensions, where n is a positive integer typed by the user.

Traffic lights

In this project, the programming of the Arduino board of 3 LED, red, yellow and green is carried out so that they turn on and off in different sequences simulating a real traffic light.

LED with photocell

In this project, the programming of the Arduino board is carried out so that an LED lights up when a photocell detects the absence of light.

Counter of 0 to 9

In this project, the programming of the Arduino board is carried out so that the 7-segment display shows the numbers from 0 to 9 with a delay of 1 second each.

Hello world on display LCD 16x2

In this project, the programming of the Arduino board is carried out so that the LCD screen shows "Hola mundo" and "Hello world".
